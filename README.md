# DialKit
DialKit SDK Distribution Package

## Install

1. In Xcode: File → Add Packages → enter:
   `https://github.com/ahmadraza738/DialKit`
2. Select the version (e.g. 1.0.1), add to your project.

## Initialize

swift
`import DialKit`

`// Your app must initialize with a license key (provided by vendor)`
`DialKit.initialize(licenseKey: "LICENSE_KEY_FROM_VENDOR") `

Download XCFramework:
https://github.com/ahmadraza738/DialKit/releases/download/1.0.0/DialKit.xcframework.zip

Current Version: 1.0.0
